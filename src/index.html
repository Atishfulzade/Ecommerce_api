API Routes

User Routes Supplier Routes Product Routes Payment Routes Order Routes Category Routes Cart Routes

User Routes

POST /api/v1/auth/login: Log in a user.

POST /api/v1/auth/register: Register a new user.

POST /api/v1/auth/logout: Log out a user.

GET /api/v1/auth/send-otp: Send OTP for verification.

POST /api/v1/auth/reset-password: Reset user password.

POST /api/v1/auth/otp: Verify OTP.

GET /api/v1/auth/profile: Show user profile.

GET /api/v1/auth/: Get all users (Admin only).

PUT /api/v1/auth/profile: Update user profile with optional image upload.

Supplier Routes

POST /login: Log in a supplier.

POST /register: Register a new supplier.

GET /profile/:id: Get supplier details by ID.

PUT /profile/:id: Update supplier details by ID.

DELETE /profile/:id: Delete supplier by ID.

GET /all: Get all suppliers (Admin only).

Product Routes

GET /: Get all products.

POST /: Create a new product.

PUT /:id: Update a product by ID.

DELETE /:id: Delete a product by ID.

GET /filter: Filter products by category.

GET /:id: Get a product by ID.

Payment Routes

POST /payment: Process a payment.

GET /payment/status/:transactionId: Get payment status by transaction ID.

GET /payment/history/:userId: Get payment history for a user by ID.

POST /payment/refund/:transactionId: Initiate a refund by transaction ID.

GET /payment/methods: Get available payment methods.

Order Routes

POST /: Create a new order.

GET /: Get all orders.

GET /:id: Get order details by ID.

PUT /:id: Update an order by ID.

DELETE /:id: Delete an order by ID.

Category Routes

GET /: Get all categories.

GET /:id: Get a category by ID.

POST /: Create a new category.

PUT /:id: Update a category by ID.

DELETE /:id: Delete a category by ID.

Cart Routes

GET /: Fetch all items in the cart.

POST /: Add a new item to the cart.

PUT /:itemId: Update a specific item in the cart.

DELETE /:itemId: Remove a specific item from the cart.
