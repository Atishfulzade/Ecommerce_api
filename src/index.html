API Project Description

This API project is designed to manage an e-commerce platform. It includes various routes for handling users, suppliers, products, payments, orders, categories, and cart functionality. The API provides secure authentication, data management, and payment processing to ensure a seamless shopping experience for both users and suppliers.

The documentation is divided into sections representing the different routes available within the API, each with its associated HTTP methods and endpoints.

User Routes

POST /api/v1/auth/login: Log in a user.

{
  "email": "user@example.com",
  "password": "yourpassword"
}

POST /api/v1/auth/register: Register a new user. 

{
  "username": "newuser",
  "firstname": "First",
  "lastname": "Last",
  "email": "user@example.com",
  "password": "yourpassword"
}

POST /api/v1/auth/logout: Log out a user.

GET /api/v1/auth/send-otp: Send OTP for verification. 

{
  "email": "user@example.com"
}

POST /api/v1/auth/reset-password: Reset user password. 

{
  "email": "user@example.com",
  "newPassword": "newpassword",
  "otp": "123456"
}

POST /api/v1/auth/otp: Verify OTP.

{
  "otp": "123456"
}

GET /api/v1/auth/profile: Show user profile.

GET /api/v1/auth/users: Get all users (Admin only).

PUT /api/v1/auth/profile: Update user profile with optional image upload. 

{
  "email": "user@example.com",
  "updatedFields": {
    "firstname": "NewFirstName",
    "lastname": "NewLastName"
  }
}

Supplier Routes

POST /api/v1/suppliers/login: Log in a supplier. 

{
  "email": "supplier@example.com",
  "password": "yourpassword"
}

POST /api/v1/suppliers/register: Register a new supplier. 

{
  "companyName": "Supplier Company",
  "contactPerson": {
    "firstname": "First",
    "lastname": "Last"
  },
  "email": "supplier@example.com",
  "password": "yourpassword",
  "vatNumber": "VAT123456"
}

GET /api/v1/suppliers/profile/:id: Get supplier details by ID.

PUT /api/v1/suppliers/profile/:id: Update supplier details. 

{
  "contactPerson": {
    "firstname": "UpdatedFirstName",
    "lastname": "UpdatedLastName"
  },
  "email": "updated@example.com"
}

GET /api/v1/suppliers: Get all suppliers (Admin only).

Product Routes

GET /api/v1/products: Get all products with optional query parameters for pagination and filtering. 

{
  "category": "Electronics",
  "priceMin": 100,
  "priceMax": 1000,
  "sortBy": "price",
  "order": "asc"
}

POST /api/v1/products: Add a new product.

{
  "name": "New Product",
  "description": "Product description here",
  "price": 299.99,
  "category": "Electronics",
  "supplierId": "supplier123"
}

GET /api/v1/products/:id: Get product details by ID.

PUT /api/v1/products/:id: Update product details. 

{
  "price": 249.99,
  "description": "Updated description"
}

DELETE /api/v1/products/:id: Delete a product.

Payment Routes

POST /api/v1/payments/charge: Process a payment. 

{
  "amount": 50.00,
  "currency": "USD",
  "paymentMethod": "card",
  "token": "token123"
}

GET /api/v1/payments/:id: Get payment details by ID.

Order Routes

POST /api/v1/orders: Create a new order.

{
  "productId": "product123",
  "quantity": 2,
  "deliveryAddress": "123 Main St"
}

GET /api/v1/orders/:id: Get order details by ID.

GET /api/v1/orders: Get all orders with optional query parameters for pagination and filtering.

PUT /api/v1/orders/:id: Update order details. 

{
  "status": "shipped"
}

DELETE /api/v1/orders/:id: Cancel an order.

Category Routes

GET /api/v1/categories: Get all categories.

POST /api/v1/categories: Add a new category. 

{
  "name": "New Category",
  "description": "Category description"
}

GET /api/v1/categories/:id: Get category details by ID.

PUT /api/v1/categories/:id: Update category details. 

{
  "name": "Updated Category Name"
}

DELETE /api/v1/categories/:id: Delete a category.

Cart Routes

GET /api/v1/cart: Get items in the cart.

POST /api/v1/cart/add: Add an item to the cart. 

{
  "productId": "product123",
  "quantity": 1
}

PUT /api/v1/cart/update: Update item quantity in the cart. 

{
  "productId": "product123",
  "quantity": 3
}

DELETE /api/v1/cart/remove: Remove an item from the cart. 

{
  "productId": "product123"
}
